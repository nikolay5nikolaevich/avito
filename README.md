# Анализ спроса на Авито

Локальный сервис: вводишь поисковый запрос — получаешь таблицу по 10 городам России с числом просмотров в день и топ-3 объявлений в каждом городе.

---

## Что нужно для работы

- **Python 3.11 или новее** — [скачать с python.org](https://www.python.org/downloads/)
  - При установке отметь галочку **«Add Python to PATH»**
- Браузер Chromium — установится автоматически на шаге 4

---

## Установка (Windows, PowerShell)

Открой PowerShell в папке проекта и выполни команды по одной.

**Шаг 1. Создай виртуальное окружение**

```powershell
python -m venv venv
```

**Шаг 2. Активируй его**

```powershell
venv\Scripts\Activate.ps1
```

> Если PowerShell выдаёт ошибку про политику выполнения, сначала запусти:
> ```powershell
> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
> ```
> Затем повтори шаг 2.

**Шаг 3. Установи зависимости**

```powershell
pip install -r requirements.txt
```

**Шаг 4. Установи браузер Chromium**

```powershell
playwright install chromium
```

---

## Запуск

```powershell
python app.py
```

Открой браузер и перейди по адресу: **http://localhost:8000**

> Виртуальное окружение должно быть активировано (шаг 2). Если закрывал терминал — активируй снова.

---

## Что делает сервис

1. Ты вводишь поисковый запрос (например, «диван» или «iPhone 14»).
2. Сервис автоматически открывает Авито в 10 городах и собирает до 150 объявлений в каждом.
3. Считает среднее число просмотров в день по городу.
4. Показывает таблицу, отсортированную по убыванию спроса.
5. Для каждого города — топ-3 объявления с разных адресов/станций метро.
6. Результаты кэшируются на 24 часа — повторный запрос тот же день будет мгновенным.
7. Можно скачать таблицу в CSV.

**Города:** Москва, Санкт-Петербург, Новосибирск, Екатеринбург, Казань, Нижний Новгород, Челябинск, Самара, Уфа, Ростов-на-Дону.

## Структура файлов

```
app.py          — веб-сервер, точка входа
parser.py       — парсинг Авито через браузер
cities.py       — список городов и их URL на Авито
selectors.py    — CSS-селекторы (правь здесь, если Авито обновил вёрстку)
analytics.py    — расчёт метрик по объявлениям
cache.py        — кэш результатов в SQLite
templates/      — HTML-шаблоны страниц
static/         — CSS и JS для интерфейса
```

## Частые вопросы

**Парсинг долго идёт** — это нормально, 15–40 минут для всех 10 городов. Прогресс отображается на странице.

**Авито заблокировал** — сервис не использует прокси в MVP. Если блокировки участятся, это будет решено в следующей версии.

**Как остановить сервер** — нажми `Ctrl+C` в терминале.
